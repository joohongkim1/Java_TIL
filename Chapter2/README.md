### Chapter 6. 배열과 ArrayList

#### 1.배열

- 동일한 자료형의 순차적 자료 구조

- 배열 선언

  ```java
  int[] arr = new int[10];
  int arr[] = new int[10];
  ```

- 메모리 구조 - 물리적 위치와 논리적 위치가 동일하다. ex) int 의 경우 1칸이 4바이트



#### 2. 다차원 배열

- 2차원 이상의 배열
- 지도, 게임, 평면이나 공간을 구현할 때 사용

- 자료형 - 배열 이름 - 행 개수 - 열 개수

  ```java
  int[][] arr = new int[2][3];
  ```



#### 3. ArrayList

- 자바에서 제공되는 객체 배열이 구현된 클래스
- 객체 배열 사용을 위한 여러 메서드들이 구현되어 있음
